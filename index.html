Navegadores web

Un navegador web (en inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web, interpretado la información de distintos tipos de archivos ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustado

Ejemplos de navegadores web

Safari: Es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS

Mozilla FIrefox es un navegador web libre y de código abierto desarrollado para Linux, ANdroid, iOS, macOS y Microsoft Windows coordinado por la corporación Mozilla y la fundación mozilla.

Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto.
